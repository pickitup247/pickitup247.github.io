2D game engine for Python running on WebAssembly

Github

About

  • 2D game engine for Python

  • written in Rust

  • compiles to WebAssembly

  • renders to WebGL

  • single binary distribution

  • no compilation

  • write an entire game in Python, or ...

Quickstart
  1. Download pyckitup binary.

Currently only linux is supported, OSX and windows should work too but it's untested.

  • Linux: download

  • Windows: build instructions

  • OS X: build instructions

  1. init
pyckitup init hello
cd hello
pyckitup

  1. Iterate over your game

  2. Once you are ready to publish, you can deploy to web with

pyckitup build
cd build
python3 server.py
